This simulation demonstrates how colour blindness affects real-world perception.

You will be shown a set of colours under different vision filters. Your task is to identify the correct original colour from distorted options.

Each level represents a different type of colour vision deficiency: Protanopia, Deuteranopia, and Tritanopia.

What this simulation represents

Real-world impact

Design takeaway

Good accessibility design avoids relying only on colour for communication. Labels, patterns, icons, contrast, and multiple feedback methods help make interfaces more inclusive for all users.
